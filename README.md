# クリプト先生 (CryptoSensei)

ユーザーが入力したニュース記事URLをもとに、記事本文を解析し、初心者向けに「要約」「重要箇所のマーカー」「補足解説」を含む解説結果を返却するAIサービス。

## サービス概要

クリプト先生は、仮想通貨に関するニュース記事を初心者にも分かりやすく解説するWebアプリケーションです。ユーザーがニュース記事のURLを入力すると、AIがその内容を解析し、以下の情報を提供します：

- **ざっくり要約**: 記事の内容を簡潔に要約
- **重要ポイント**: 記事内の重要な情報をハイライトし、詳細な解説と分かりやすい例え話を提供
- **読み飛ばし可能な部分**: 初心者が理解する上で重要度の低い部分を指摘

## 技術スタック

### フロントエンド
- **フレームワーク**: React 18.3.1
- **言語**: TypeScript 5.5.3
- **UIライブラリ**:
  - Tailwind CSS 3.4.1 (スタイリング)
  - Lucide React 0.344.0 (アイコン)
- **ビルドツール**: Vite 5.4.2

### バックエンド
現在のデモ実装ではフロントエンドのみで動作しており、モックデータを使用しています。本番環境では以下のような実装が想定されます：

- API サーバー: 記事URLを受け取り、解析結果を返すエンドポイント
- Web スクレイピング: 記事URLからコンテンツを取得する機能
- AI 解析: 記事内容を解析し、要約・重要ポイント・読み飛ばし部分を特定する機能

## 主要機能

1. **URL入力**: ユーザーが仮想通貨関連のニュース記事URLを入力
2. **AI解析**: 入力されたURLの記事内容をAIが解析
3. **要約表示**: 記事の要点を簡潔に表示
4. **重要ポイントのハイライト**: 重要度に応じて★マークで表示し、詳細な解説と例え話を提供
5. **音声読み上げ**: 要約や重要ポイントを音声で聞くことが可能
6. **履歴機能**: 過去に解析した記事の履歴を保存・閲覧可能
7. **共有機能**: 解析結果を他の人と共有可能

## UI/UXの特徴

- **レスポンシブデザイン**: モバイル・デスクトップの両方に対応
- **カラーテーマ**: ブルーを基調としたシンプルで読みやすいデザイン
- **カード型レイアウト**: 情報を整理して表示
- **アイコン活用**: 視覚的に分かりやすいアイコンを使用
- **音声読み上げ**: アクセシビリティ向上のための音声機能

## データフロー

1. ユーザーがニュース記事のURLを入力
2. フロントエンドがバックエンドAPIにURLを送信
3. バックエンドが記事をスクレイピングして内容を取得
4. AIが記事内容を解析し、要約・重要ポイント・読み飛ばし部分を特定
5. 解析結果がフロントエンドに返され、ユーザーに表示
6. 結果はローカルストレージに保存され、履歴として利用可能
